ClaimDesk helps WooCommerce store owners manage product return and exchange claims through a guided multi-step claim submission process.

== Description ==

ClaimDesk is a WooCommerce claim management plugin that allows customers to submit return or exchange requests easily while giving store owners a structured dashboard to manage claims efficiently.

The plugin provides a user-friendly guided workflow for customers to submit claims and a centralized admin interface inside WooCommerce to manage and track requests.

=== Compatibility ===

* WordPress: 5.0 or higher  
* PHP: 7.4 or higher  
* WooCommerce: 3.0 or higher  
* Fully compatible with WooCommerce HPOS (8.x / 9.x)

=== Key Features ===

✔ Multi-step claim submission workflow  
✔ Customers can select products and quantity for claim  
✔ Supports Return and Exchange claim types  
✔ Customers can select problem/reason for claim  
✔ Customers can describe issue details  
✔ Image upload support for damage proof  
✔ Refund method selection option  
✔ Claim review screen before final submission  
✔ Dedicated claim management dashboard inside WooCommerce  
✔ Fully configurable admin settings  
✔ Built specifically for WooCommerce stores  

=== Claim Submission Flow ===

Step 1 – Select Product  
Customers select purchased product and quantity to return or exchange.

Step 2 – Claim Details  
Customers can:
- Select claim type (Return or Exchange)
- Select problem or reason
- Describe the issue
- Upload supporting images
- Choose refund method

Step 3 – Review & Submit  
Customers review their claim before final submission.

=== Admin Features ===

Store administrators can:

- View all submitted claims
- Manage claim status
- Review customer uploaded evidence
- Configure plugin settings
- Manage claim workflow from WooCommerce dashboard

=== Future Roadmap ===

Future versions may include AI-powered assistance that helps customers select claim reasons dynamically based on product type and claim patterns.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure WooCommerce is installed and activated.
4. Configure plugin settings from WooCommerce → ClaimDesk.

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =
Yes. ClaimDesk works only with WooCommerce.

= Can customers upload images as proof? =
Yes, customers can upload images while submitting claims.

= Does the plugin support both return and exchange requests? =
Yes, the plugin supports both claim types.

= Is the plugin compatible with WooCommerce HPOS? =
Yes, ClaimDesk is fully compatible with WooCommerce HPOS (High-Performance Order Storage).

= Is there a premium version available? =
Currently, ClaimDesk is fully free. A Pro version may be released in future with additional features.
